# 🚀 Инструкция по деплою — Кибертроеборье

## Часть 1: Настройка Google Таблицы и Apps Script

### Шаг 1 — Создай Google Таблицу

1. Перейди на [sheets.google.com](https://sheets.google.com)
2. Нажми **«+»** — создай новую таблицу
3. Дай название: `Кибертроеборье — Заявки`

### Шаг 2 — Открой редактор Apps Script

1. В таблице: **Расширения → Apps Script**
2. Откроется редактор кода

### Шаг 3 — Вставь скрипт

1. Удали весь существующий код (там будет пустая функция)
2. Скопируй **весь** код из файла `google-apps-script.gs`
3. Вставь в редактор
4. Нажми **💾 Сохранить** (или Ctrl+S)

### Шаг 4 — Разверни как веб-приложение

1. Нажми **«Развернуть»** (синяя кнопка справа вверху)
2. Выбери **«Новое развёртывание»**
3. Нажми на шестерёнку ⚙️ → выбери тип **«Веб-приложение»**
4. Настройки:
   - **Описание**: `Кибертроеборье API v1`
   - **Выполнять как**: Я (my@gmail.com)
   - **Доступ**: **Все** (включая анонимных)
5. Нажми **«Развернуть»**
6. Если попросит разрешения — нажми **«Разрешить»** и авторизуй
7. Скопируй URL вида:
   ```
   https://script.google.com/macros/s/AKfycb.../exec
   ```

### Шаг 5 — Вставь URL в код сайта

1. Открой файл `src/App.jsx`
2. Найди строку:
   ```js
   const SCRIPT_URL = 'https://script.google.com/macros/s/ВСТАВЬ_СВОЙ_URL_ЗДЕСЬ/exec'
   ```
3. Замени `ВСТАВЬ_СВОЙ_URL_ЗДЕСЬ` на твой реальный ID скрипта
4. Сохрани файл

> ⚠️ **Важно**: При каждом изменении скрипта нужно создавать **новое развёртывание** — старый URL не обновляется автоматически.

## Часть 2: Деплой на Vercel

### Шаг 1 — Создай репозиторий на GitHub

```bash
# В папке проекта clown-cup_v2 выполни:
git init
git add .
git commit -m "feat: Кибертроеборье v1 🏆"
```

1. Перейди на [github.com/new](https://github.com/new)
2. Название: `clown-cup`
3. Нажми **«Create repository»**
4. Выполни команды, которые покажет GitHub:
   ```bash
   git remote add origin https://github.com/ТВО_ЛОГИН/clown-cup.git
   git branch -M main
   git push -u origin main
   ```

### Шаг 2 — Подключи к Vercel

1. Перейди на [vercel.com](https://vercel.com) и войди через GitHub
2. Нажми **«Add New... → Project»**
3. Найди репозиторий `clown-cup` → нажми **«Import»**
4. Настройки (обычно определяются автоматически):
   - **Framework Preset**: Vite
   - **Build Command**: `npm run build`
   - **Output Directory**: `dist`
5. Нажми **«Deploy»**

### Шаг 3 — Готово!

Vercel даст тебе ссылку вида `https://clown-cup-xxxxxxx.vercel.app` 🎉

При пуше новых изменений в `main` Vercel автоматически пересоберёт сайт.

## Часть 3: Переменные окружения (опционально)

Если хочешь не хардкодить URL скрипта в коде, можно использовать env-переменные:

1. В Vercel: **Settings → Environment Variables**
2. Добавь: `VITE_SCRIPT_URL` = `https://script.google.com/macros/s/.../exec`
3. В `src/App.jsx` измени строку на:
   ```js
   const SCRIPT_URL = import.meta.env.VITE_SCRIPT_URL || ''
   ```

## FAQ

**Q: Форма работает, но данные не попадают в таблицу?**
A: Проверь, что в `SCRIPT_URL` вставлен правильный URL. При первом запросе Apps Script может попросить разрешения.

**Q: CORS ошибка в консоли?**
A: Это нормально при первом запуске. Убедись что в настройках развёртывания выбрано **«Доступ: Все»**.

**Q: Хочу поменять третью игру?**
A: В `src/App.jsx` найди упоминания `wormix` в полях формы и в карточках команд — замени на нужную игру.
